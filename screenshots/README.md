# Screenshots

This directory contains screenshots for the plugin README and WordPress.org listing.

## Required Screenshots

1. **dashboard.png** - Main dashboard view
2. **backups.png** - Backup management interface
3. **settings.png** - Settings and configuration page
4. **restore.png** - Restore interface

## Screenshot Guidelines

- **Size**: 1200x900 pixels (4:3 ratio)
- **Format**: PNG
- **Quality**: High quality, optimized for web
- **Content**: Show actual plugin interface
- **Annotations**: Optional, but helpful for complex features
- **Browser**: Use a modern browser with clean UI

## Taking Screenshots

1. Use WordPress admin in a clean environment
2. Use consistent browser and window size
3. Hide sensitive information (credentials, etc.)
4. Use placeholder data for demonstrations
5. Ensure UI is clean and professional

## Tools

- **Screenshot Tools**:
  - macOS: Cmd+Shift+4
  - Windows: Snipping Tool
  - Browser extensions: Full Page Screen Capture
- **Editing**:
  - Photoshop, GIMP, or similar
  - Online tools: Canva, Figma
