## Screenshot Guidelines

- **Size**: 1200x900 pixels (4:3 ratio) recommended
- **Format**: PNG
- **Quality**: High quality, optimized for web
- **Content**: Show actual plugin interface with real data
- **Annotations**: Optional, but helpful for complex features
- **Browser**: Use a modern browser with clean UI (Chrome, Firefox, Safari)

## Taking Screenshots

1. Use WordPress admin in a clean environment
2. Use consistent browser and window size (1920x1080 or larger)
3. Hide sensitive information (credentials, API keys, etc.)
4. Use placeholder data for demonstrations
5. Ensure UI is clean and professional
6. Show both optimized and unoptimized images in media gallery
7. Include optimization statistics in screenshots
8. Show modal dialogs when relevant (optimization modal, comparison modal)

## Tools

- **Screenshot Tools**:
  - macOS: Cmd+Shift+4 (selection) or Cmd+Shift+3 (full screen)
  - Windows: Snipping Tool or Win+Shift+S
  - Browser extensions: Full Page Screen Capture, Awesome Screenshot
  - Professional: Snagit, Lightshot
- **Editing**:
  - Photoshop, GIMP, or similar
  - Online tools: Canva, Figma
  - Annotation: Annotate, Skitch
