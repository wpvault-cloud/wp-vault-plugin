# Plugin Assets

This directory should contain the following WordPress plugin assets:

## Required Assets

### Plugin Icon

- **icon-256x256.png** - Main plugin icon (256x256 pixels)
- **icon-128x128.png** - Smaller icon variant (128x128 pixels)

### Banner Images (for WordPress.org)

- **banner-772x250.png** - Plugin banner (772x250 pixels)
- **banner-1544x500.png** - High-resolution banner (1544x500 pixels)

### Screenshots (for WordPress.org)

- **screenshot-1.png** - Dashboard view
- **screenshot-2.png** - Backup management
- **screenshot-3.png** - Settings page
- **screenshot-4.png** - Restore interface

## Asset Guidelines

- **Format**: PNG with transparency where appropriate
- **Colors**: Use brand colors consistently
- **Style**: Modern, clean, professional
- **Text**: Minimal text, focus on visual representation
- **Optimization**: Compress images for web (use tools like TinyPNG)

## Creating Assets

You can use the SVG icon from the landing page (`wp-vault-landing-page/public/icon.svg`) as a base and export it in the required sizes.

For screenshots, take actual screenshots of the plugin interface in WordPress admin.
